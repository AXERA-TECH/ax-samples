# AX620A

## Brief

- CPU
    - Quad-core Cortex A7
- NPU
    - 7.2Tops@INT4 or 1.8Tops@INT8
- ISP
    - 4K@30fps
    - AI-ISP
- Video Encoder
    - H.264/H.265, 4K@30fps+
- Video Decoder
    - H.264, 1080p@60fps
- Video Interfaces
    - Video Input Interfaces
        - 3x4 Lane MIPI CSI
    - Video Output Interfaces
        - 2x4 Lane MIPI CSI/DSI
- DDR
    - 32bit LPDDR4x(x)
    - Up to 3733Mbit/s rate
- Ethernet
    - One Gigabit Ethernet port
    - RGMII and RMII modes
